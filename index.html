About Me

Hello! My name is Khensani Ruth Mahwayi, and I’m an enthusiastic Software development beginner. This page marks my first step into building accessible, well-structured websites using only HTML5. I believe in learning web fundamentals the right way— starting with clean, semantic code.

What I’ve Learned

Semantic HTML5

I’ve learned that tags like <section>, <article>, and <header> give content meaning — making it easier for both people and search engines to understand.

Accessibility

Good structure helps users with screen readers navigate my page. Using proper heading levels and ARIA labels supports inclusive design from the start.

SEO Basics

Clear titles, metadata, and meaningful headings make it easier for search engines to index and rank pages. I’m learning how structure improves visibility.

Projects

This very page is my first project — built completely with HTML5. I used no styling or scripts, just structure and content. Future projects will include a personal portfolio and simple applications as I learn more.
